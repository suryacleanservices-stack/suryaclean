## 🔑 Features
- **Google Sign-In** — users login with existing Google account
- **UPI Payments** — via Razorpay (GPay, PhonePe, Paytm)
- **Slot Booking** — pick date/time for one-time cleaning
- **AMC Subscriptions** — Basic/Standard/Premium annual plans
- **Order History** — complete booking + transaction records
- **Admin Dashboard** — assign workers, track revenue
- **Android + iOS** — single Flutter codebase
