Step 2: Setup Firebase (Google Login + Database)

⏱ 15 minFree
▼

Firebase is Google's backend platform. It gives you Google Sign-In for users and Firestore to store bookings, AMCs, and payments.

📋 What You'll Get From This Step

  • A firebaseConfig object → goes into website environment variables
  • An Admin SDK private key JSON → goes into backend environment variables
  • A google-services.json → goes into Flutter Android app
  • A GoogleService-Info.plist → goes into Flutter iOS app

3.1 Create Firebase Project

  1. Go to console.firebase.google.com
  2. Click "Add Project" → Name: suryaclean
  3. Enable Google Analytics (recommended — free)
  4. Select your Google account → Create Project

3.2 Enable Google Sign-In

  1. Left sidebar → Build → Authentication → "Get Started"
  2. Sign-in method tab → Click Google
  3. Toggle Enable → Select your support email → Save

3.3 Enable Firestore Database

  1. Left sidebar → Build → Firestore Database → "Create Database"
  2. Choose "Start in test mode"
  3. Location: asia-south1 (Mumbai) — closest to Faridabad!

3.4 Register Web App (for Website)

  1. Project Overview → Click the Web icon
  2. Nickname: SuryaClean Web → Register
  3. COPY the firebaseConfig object — save it for Step 5!

3.5 Register Android App (for Flutter)

  1. Project Overview → Click the Android icon 🤖
  2. Package name: com.suryaclean.app
  3. Download google-services.json — save to mobile/android/app/

3.6 Generate Admin SDK Key (for Backend)

⚠️ Keep This File SECRET — Never Upload to GitHub

  1. Project Settings → Service accounts tab
  2. Click "Generate new private key"
  3. Download the JSON file — it contains your client_email and private_key
💳

Step 3: Setup Razorpay (UPI Payments)

⏱ 3-5 business days (KYC)Free to sign up
▼

Razorpay handles all payments — UPI (GPay, PhonePe), cards, and netbanking. UPI payments are 0% fee!

Sign Up & Verify

  1. Go to dashboard.razorpay.com
  2. Sign up with hello@suryaclean.in
  3. Business: SuryaClean | Type: Service | Category: Cleaning Services
  4. Submit KYC documents: PAN Card, Aadhaar, Bank account, Canceled cheque
  5. Wait 3-5 business days for verification → your account switches to Live Mode

Get Your Keys

  1. Dashboard → Settings → API Keys
  2. Key ID (public, starts with rzp_live_) → goes into website + app
  3. Key Secret (PRIVATE!) → goes into backend ONLY

⚠️ CRITICAL: Key Secret NEVER Goes in Frontend Code

Key ID = public (okay in website/app). Key Secret = private (backend only). If the secret leaks, anyone can refund your payments.

Step 4: Deploy Backend API (Render)

⏱ 15 minFree
▼

The backend handles all business logic — creating bookings, processing payments, and managing users. We deploy it on Render (free tier).

Push Code to GitHub First

💻 Run these commands in terminal:

cd /home/jkaushik/Solar/suryaclean-app/backend
git init
git add .
git commit -m "Initial SuryaClean backend"

# Create a repo on github.com, then:
git remote add origin https://github.com/YOUR_USERNAME/suryaclean-backend.git
git push -u origin main

Deploy on Render

  1. Go to render.com → Sign up with GitHub
  2. Click "New +" → "Web Service"
  3. Connect repo: suryaclean-backend
  4. Configure:
    • Name: suryaclean-api
    • Runtime: Node
    • Build Command: npm install
    • Start Command: node index.js

🔑 Add Environment Variables

In Render → Environment tab, add these variables using the values from Steps 2 & 3:

FIREBASE_PROJECT_ID=suryaclean
FIREBASE_CLIENT_EMAIL=firebase-adminsdk-xxxxx@your-project.iam.gserviceaccount.com
FIREBASE_PRIVATE_KEY=-----BEGIN PRIVATE KEY-----\nYOUR_KEY_HERE\n-----END PRIVATE KEY-----
FIREBASE_DATABASE_URL=https://suryaclean.firebaseio.com
RAZORPAY_KEY_ID=rzp_live_xxxxxxxxxx
RAZORPAY_KEY_SECRET=your_live_razorpay_secret
PORT=5000
NODE_ENV=production
ALLOWED_ORIGINS=https://suryaclean.in,https://www.suryaclean.in
  1. Click "Create Web Service"
  2. ✅ Your API is live at: https://suryaclean-api.onrender.com

💡 Free Tier Note

Render's free tier puts your service to sleep after 15 minutes of no traffic. First request takes ~30 seconds to wake up. To keep it always awake, either:

  • Upgrade to $7/month plan (recommended when you have real customers)
  • Or setup a free cron job at cron-job.org to ping every 10 minutes

Step 5: Deploy Website (Vercel)

⏱ 10 minFree
▼

Vercel is built by the creators of Next.js — it's the fastest, easiest way to deploy your website. 100% free for the traffic you'll get initially.

Push Code to GitHub

cd /home/jkaushik/Solar/suryaclean-app/web
git init
git add .
git commit -m "Initial SuryaClean website"
git remote add origin https://github.com/YOUR_USERNAME/suryaclean-web.git
git push -u origin main

Deploy on Vercel

  1. Go to vercel.com → Sign up with GitHub
  2. Click "Add New" → "Project"
  3. Import repo: suryaclean-web
  4. Framework: Next.js (auto-detected)

🔑 Add Environment Variables

In Vercel → Environment Variables, paste these:

NEXT_PUBLIC_FIREBASE_API_KEY=AIzaSy...        ← from Firebase Step 3.4
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=suryaclean.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=suryaclean
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=suryaclean.appspot.com
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=123456789
NEXT_PUBLIC_FIREBASE_APP_ID=1:123456789:web:abc123
NEXT_PUBLIC_API_URL=https://suryaclean-api.onrender.com   ← from Step 4
NEXT_PUBLIC_RAZORPAY_KEY_ID=rzp_live_xxxx                 ← from Step 3
  1. Click "Deploy"
  2. ✅ Website is LIVE at: https://suryaclean-web.vercel.app
🔗

Step 6: Connect Domain to Website

⏱ 5 min setup + 1-60 min DNS propagation
▼

Your website is live at a Vercel URL, but you want it on suryaclean.in.

In Vercel:

  1. Vercel Dashboard → Your project → Settings → Domains
  2. Add: suryaclean.in
  3. Vercel shows you DNS records to add

At Your Domain Registrar (Namecheap/GoDaddy):

  1. Find DNS Settings / Advanced DNS
  2. Add these records:
TypeNameValue
A@76.76.21.21 (Vercel's IP)
CNAMEwwwcname.vercel-dns.com
  1. Wait 5-60 minutes → Visit https://suryaclean.in

🎯 Bonus: Custom Backend Subdomain

Make your backend professional by putting it on api.suryaclean.in:

  1. In Render → Settings → add Custom Domain: api.suryaclean.in
  2. Add a CNAME record in your DNS: api → your-render-url.onrender.com
  3. Update website env: NEXT_PUBLIC_API_URL=https://api.suryaclean.in
